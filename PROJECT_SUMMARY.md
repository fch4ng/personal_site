## üìù Customization Quick Reference

### Change Colors
Edit `style.css` lines 8-12:
```css

--text-color: #ffffff;    /* Text */
--accent-color: #00ff00;  /* Hover effects */
```

1. Add HTML in `index.html`
2. Style in `style.css`
3. Add logic in `script.js` (if needed)
